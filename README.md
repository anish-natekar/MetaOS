MetaOS is a simple Agent built on concepts from natural language and operating system.

Goal is to define a simple meta (higher level of abstraction) level Instruction Set consisting Operationg based on Certain words from the English Language.

These Instruction serves as the tools for our LLM Agent to manage tasks in a more scalable way. Agent should be able to piece these operations from the instruction in it's instruction memory and should be able to execute these tasks efficiently withing the llm api limitation (context window, rate limits, etc).

The agent should be able to manage and search through it's own memory that learns over-time and pull memories based on the goal and requirements.

some aspects of this project maybe made into youtube videos for the sake of learning. the code will be open on Github for anyone to try, learn, discuss, and contribute. A github webpage will be maintained with relevant documentation and other instructions.

My hope is to gain more clarity about autonomous agents by building a autonomous agent from the ground up.
